Propostas de Utilização de Sistemas OCR para o Desenvolvimento de um assistente virtual para deficientes visuais

Sobre

A aplicação de recursos tecnológicos no âmbito da inclusão e acessibilidade à pessoas com necessidades especiais é um campo de importância indiscutível para pesquisadores em geral.

Nesse sentido, torna-se relevante desenvolver dispositivos que possibilitem a deficientes visuais o acesso a arquivos de texto, que é a motivação deste projeto. Smart Braille se trata da iniciativa de sanar ou diminuir a escassez das dificuldades que um deficiente visual tem para ler textos.

A abordagem deste projeto destina-se em duas propostas: uma aplicação em Android para reconhecimento de caracteres, e um computador de baixo custo para iniciar um script em python para o reconhecimento de caracteres usando uma webcam.
